Quick Start

Ikuti 3 langkah simple ini untuk mulai pakai API:

  1. Dapatkan API Key

    Chat WhatsApp kami untuk mendapatkan API key gratis. Respon dalam 5 menit!

    Chat WhatsApp
  2. Test API Key

    Coba endpoint sederhana untuk memastikan API key kamu berfungsi:

    curl "https://bangbotz-apis.vercel.app/api/test/apikey?apikey=YOUR_API_KEY"
  3. Mulai Coding!

    Integrasikan ke bot kamu. Lihat code examples di bawah.

‚úÖ Pro Tip

Simpan API key kamu di environment variable, jangan hardcode di code!

Authentication

Semua request harus include API key sebagai query parameter apikey. 

GET https://bangbotz-apis.vercel.app/api/downloader/youtube?apikey=YOUR_API_KEY&url=...

‚ö†Ô∏è Security Warning

Jangan share API key kamu ke orang lain. Kalau bocor, segera regenerate di dashboard atau contact support.

Rate Limits

Rate limit berbeda per tier untuk menjaga stabilitas server:

Tier Requests/Month Requests/Minute
Free 1,000 10
Premium 50,000 60
VIP Unlimited No limit

üìä Checking Your Quota

Setiap response includes quota info di field quota. Monitor usage kamu real-time!

Error Codes

API menggunakan standard HTTP status codes:

Code Meaning Action
200 Success Request berhasil
400 Bad Request Cek parameter yang dikirim
401 Unauthorized API key invalid atau missing
429 Too Many Requests Rate limit exceeded, tunggu sebentar
500 Server Error Server issue, coba lagi atau contact support
401 Error Response Example 
{
  "status": false,
  "message": "Invalid API Key"
}

YouTube Downloader

Download video & audio dari YouTube dengan kualitas terbaik.

GET
/api/downloader/youtube

Parameters

Parameter Type Required Description
apikey string Required Your API key
url string Required YouTube video URL

Request Example

curl "https://bangbotz-apis.vercel.app/api/downloader/youtube?apikey=bbz-xxx&url=https://youtube.com/watch?v=dQw4w9WgXcQ"
const axios = require('axios');

const apikey = 'bbz-xxx';
const url = 'https://youtube.com/watch?v=dQw4w9WgXcQ';

axios.get('https://bangbotz-apis.vercel.app/api/downloader/youtube', {
  params: { apikey, url }
})
.then(response => {
  console.log(response.data);
  // Access video URL
  const videoUrl = response.data.data.video.url;
})
.catch(error => {
  console.error(error.response.data);
});
import requests

apikey = 'bbz-xxx'
url = 'https://youtube.com/watch?v=dQw4w9WgXcQ'

response = requests.get(
    'https://bangbotz-apis.vercel.app/api/downloader/youtube',
    params={'apikey': apikey, 'url': url}
)

data = response.json()
print(data)

# Access video URL
video_url = data['data']['video']['url']

Integration Examples

Contoh integrasi ke berbagai platform bot WhatsApp:

Baileys (Node.js)

const { default: makeWASocket } = require('@whiskeysockets/baileys');
const axios = require('axios');

// Handler untuk command !ytdl
sock.ev.on('messages.upsert', async ({ messages }) => {
  const msg = messages[0];
  const text = msg.message?.conversation || '';
  
  if (text.startsWith('!ytdl ')) {
    const url = text.replace('!ytdl ', '');
    
    // Hit API
    const response = await axios.get('https://bangbotz-apis.vercel.app/api/downloader/youtube', {
      params: {
        apikey: process.env.BANGBOTZ_API_KEY,
        url: url
      }
    });
    
    const data = response.data.data;
    
    // Send video
    await sock.sendMessage(msg.key.remoteJid, {
      video: { url: data.video.url },
      caption: `‚úÖ ${data.title}\n\n` +
               `üë§ ${data.author}\n` +
               `‚è±Ô∏è ${data.duration}s\n` +
               `üìä ${data.views} views`
    });
  }
});

Python (yowsup)

import requests
import os

def handle_message(message):
    if message.startswith('!ttdl '):
        url = message.replace('!ttdl ', '')
        
        # Hit API
        response = requests.get(
            'https://bangbotz-apis.vercel.app/api/downloader/tiktok',
            params={
                'apikey': os.getenv('BANGBOTZ_API_KEY'),
                'url': url
            }
        )
        
        data = response.json()['data']
        
        # Download & send video
        video_url = data['video']['noWatermark']
        send_video(video_url, caption=data['title'])
